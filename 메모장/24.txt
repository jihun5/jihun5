통계학이란? 
1. 데이터(자료) - 전통적인 데이터 수집 방법(관찰, 설문조사, 실험 등)
	       - 하나 이상의 변수에 대한 관찰값의 모음
	       - 열 단위로 변수라고 정의
	       - columns 은 변수
2. 자료를 가공해 얻는 것이 정보
3. 통계학 - 데이터에서 쓸모 있는 정보를 얻기 위한 별도의 과정
	-불확실한 현상을 이해하기 위해 데이터를 수집하고, 데이터 패턴을 요약, 분석하여 불확실한 현상에 대한 결론
	을 찾는 학문
데이터 수집 - 총조사/전수조사(cencus)  
	    - 표본조사(모집단 내에서 그 집단의 특성을 잘 나타낼 수 있는 일부를 추출)
데이터 요약 - 데이터가 가진 특징과 패턴을 정확하고 효과적으로 드러내기 위한 통계적 방법을 사용: 기술통계(평균,중간값,최소값 등등)
추론 - 데이터를 이용하여 추측하고 그 추측의 신뢰성을 계량화: 추측통계(추론통계)
데이터 - 단위 : 관측되는 개별 대상
          - 변수:  각 단위에 대해 관측되는 특성
          - 관찰값: 각 단위로부터 관측한 특성의 값
모집단(전체) - *모집단의 평균(모수)
표본 - 모집단의 일부로서 모집단을 잘 반영하는 표본을 뽑는 것은 매우 중요
       - 표본오차: 모집단을 대표할 수 있는 표본 단위들이 조사대상으로 추출되지 못함으로서 발생하는 오차
표본추출방법 - 단순랜덤표집(모집단에서 n명 추출, 랜덤으로), 비복원(뽑힌 친구는 다시 X) 복원 추출(추출한 요소를 다시 집어 넣어추출)
	     - 계통추출법(N번씩 사람뽑기)
	     - 집락추출법(군집을 구분하고, 단순랜덤 추출법, 지역표본추출)
  	     - 층화추출법(유사한 원소끼리 몇 개의 층 (stratum)으로 나누어 각 층에서 랜덤 추출)
통계량 - 표본의 특성을 나타내는 대표값 / 모수를 추정하기 위해 표본에서 얻은값
모수는 우리가 알고싶은 모집단의 특성을 나타내는 대푯값이고, 모수를 알기 위해 표집한 표본의 특성을
나타내는 대푯값을 통계량이라고 한다.

측정 = 표본조사나 실험을 하는 과정에서 추출된 원소나 관측자료를 얻는 것
변수(variable) = Data (각 단위(한 행)에 대해 관측되는 특성(열))

데이터 전처리 - 분석에 적합한 형태로 데이터를 가공하는 것
	- legacy : 원래가지고 있던 데이터 예) 마케팅, 세일즈, ERP, CMS
	- staging : 데이터를 가공 없이 로딩 하는 영역 예) 비정형 데이터
	- ODS : 로딩된 데이터를 DW에 저장하기 전에 임시로 저장하는 영역 예) 비정형 —> 정형 데이터로 처리
	- 데이터웨어하우스(DW) : 데이터를 주제별로 통합하여, 별도의 장소에 저장해 놓은 통합 데이터베이스.
	- 데이터마트(DM) : 분석의 편의성을 높이고자, DW의 데이터를 주제별, 업무별로 요약하여 구성한 데이터저장소. 
			일반적으로 각 부서별로 다양한 예측과 분석을 목표로 만들어진다.
			예) DataFrame

활용 - 데이터웨어하우스(DW), 데이터마트(DM)를 통해 분석 데이터를 가져와 사용
최종 데이터 구조로 가공 - 분류값과 입력 변수들을 연관시켜 인구통계, 요약변수, 파생변수 등을 산출
		- 비정형 데이터 : DBMS에 저장됐다가 텍스트 마이닝을 거쳐 데이터 마트와 통합
		- 관계형 데이터 : DBMS에 저장되어 사회 신경망분석을 거쳐 분석 결과 통계값이 마트와 통합되어 활용
데이터 분석(=통계) - 특정집단을 대상으로 수행한 조사나 실험을 통해 나온 결과에 대한 요약된 형태의 표현
데이터 분석 구분 
1)  탐색적 데이터 분석(EDA) -  기술통계
			- 데이터를 요약해 설명하는 기법
			- 데이터 수집 > 시각화 탐색 > 패턴 도출 > 인사이트 발견
			- 사람들이 받는 월급을 집계해 전체 월급 평균을 구한다.
2) 확증적 데이터 분석 (CDA) - 추론통계
			- 단순히 숫자를 '요약'하는 것을 넘어 어떤 값이 발생할 확률을 계산하는 통계 기법
			- 모집단에서 샘플링한 표본을 가지고 모집단의 특성을 추론하고 그 결과가 신뢰성이 있는지 검정하는 것이다.
			- 가설 설정 > 데이터 수집 > 탐색적 데이터 분석 > 추론통계 > 가설검증
			- 수집된 데이터에서 성별에 따라 월급에 차이가 있는 것으로 나타났을 때 이런 차이가 우연히 발생할 확률을 계산한다.

탐색적 탐색적 데이터 분석(EDA)
특이한 점이나 의미있는 사실을 도출
데이터의 특징과 내재하는 구조적 관계를 알아내기 위한 기법들의 통칭
기술통계 - 모집단으로부터 표본을 추출하고 표본이 가지고 있는 정보를 쉽게 파악할 수 있도록 
	데이터를 정리하거나 요약하기 위해 하나의 숫자 또는 그래프의 형태로 표현하는 절차
	- Sample에 대한 특성인 평균，표준편차 , 중위수，최빈값，그래프 , 왜도，첨도 등을 구하는 것을 의미
탐색적 데이터의 분석의 과정
데이터이해 단계  - 변수의 분포와 특성 파악
변수생성 단계 - 분석 목적에 맞도록 데이터 요약 및 파생변수 생성
변수선택 단계 - 목적변수에 의미있는 후보 변수 선택
데이터 시각화 - 데이터의 상태를 한눈에 확인할 수 있도록 요약하여 그래픽으로 표현
확증적 데이터 분석 (CDA)
추론통계 - 실질적인 데이터 분석(통계)를 의미하는 과정
	- 자료의 정보를 이용해 집단에 관한 추측, 결론을 이끌어내는 과정
	- 표본을 통해 모집 단을 추정하는 것을 의미
데이터마이닝(머신러닝) - 대용량의 자료로부터 정보를 요약
		     - 관계, 패턴, 규칙 등을 탐색
		     - 미래에 대한 예측

기계학습 - 컴퓨터가 학습할 수 있도록 알고리즘과 기술을 개발하는 분야
